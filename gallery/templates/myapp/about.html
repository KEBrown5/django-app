About The Blog

The Blog is a personal project developed by Kyle Brown. Its an app dedicated to giving users an outlet to write store, and manage their blog posts.

It demonstrates many aspects of web-development, ranging from UI design, database management, CRUD functionality, authentication and security, and RESTful API development.
